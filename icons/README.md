# Icons

This directory contains icon assets organized by collection or project.

## Structure

```
icons/
├── phosphor/          # Phosphor icon collection (multi-platform)
├── scripts/           # Generation and optimization scripts
│   └── phosphor/     # Phosphor-specific scripts (React, Swift, Web)
└── README.md          # This file
```

## Usage

Icons in this library can be accessed remotely across projects. Each subdirectory represents a specific icon collection or project.

## Adding Icons

1. Place icon files in the appropriate collection directory (e.g., `phosphor/`)
2. Use consistent naming conventions
3. Include metadata or documentation as needed
4. Update this README if adding new collections

## Collections

### phosphor
Phosphor icon collection - optimized for multi-platform use (web, React, Swift). SVG files work natively with design tools like Figma and Sketch. 

- **Documentation**: See [phosphor/README.md](./phosphor/README.md) for platform-specific usage
- **Scripts**: See [scripts/phosphor/README.md](./scripts/phosphor/README.md) for automation scripts (React components, Swift assets, SVG optimization)
